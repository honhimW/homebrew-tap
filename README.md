# Homebrew Tap

## How do I install these formulae?

```shell
brew install honhimW/tap/ratisui
```

**Or**
```shell
brew tap honhimW/tap
brew install ratisui
```

**Or**
in a `brew bundle` `Brewfile`:

```ruby
tap "honhimW/tap"
brew "ratisui"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
